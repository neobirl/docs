# Função Main

{% hint style="info" %}
Este é um projeto humorístico. Não leve tudo o que for dito aqui a sério (ou leve, problema é seu).
{% endhint %}

Para declarar a função main em BIRL, basta criarmos:

{% code title="1 - 1" %}
```
HORA DO SHOW
//código
BIRL
```
{% endcode %}

`HORA DO SHOW` é equivalente a `int main (void){` e `BIRL` é equivalente a `}`. Isso faz com que o código `1 - 1` em BIRL seja equivalente a o seguinte código em C:

{% code title="1 - 2" %}
```c
int main(void){

}
```
{% endcode %}

Para retornarmos algo na função, basta acrescentarmos a palavra reservada (ou palavras reservadas? me deixou confuso) `BORA CUMPADE <valor>`, que é equivalente a `return` em C.&#x20;

{% code title="1 - 3" %}
```
HORA DO SHOW
 BORA CUMPADE 0;
BIRL
```
{% endcode %}
